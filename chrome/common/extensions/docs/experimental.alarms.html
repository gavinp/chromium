API reference: chrome.experimental.alarms

Methods

clear

chrome.experimental.alarms.clear(string name)

Clears the alarm with the given name.

Parameters

name
( optional string )
The name of the alarm to clear. Defaults to the empty string.

clearAll

chrome.experimental.alarms.clearAll()

Clears all alarms.

create

chrome.experimental.alarms.create(string name, object alarmInfo)

Creates an alarm. After the delay is expired, the onAlarm event is fired. If there is another alarm with the same name (or no name if none is specified), it will be cancelled and replaced by this alarm.

Parameters

name
( optional string )
Optional name to identify this alarm. Defaults to the empty string.
alarmInfo
( object )
Undocumented.
delayInSeconds
( integer )
Length of time in seconds after which the onAlarm event should fire. Note that granularity is not guaranteed: this value is more of a hint to the browser. For performance reasons, alarms may be delayed an arbitrary amount of time before firing.
repeating
( optional boolean )
True if the alarm should repeatedly fire at regular intervals. Defaults to false.

get

chrome.experimental.alarms.get(string name, function callback)

Retrieves details about the specified alarm.

Parameters

name
( optional string )
The name of the alarm to get. Defaults to the empty string.
callback
( function )
Undocumented.

Callback function

The callback parameter should specify a function that looks like this: 

function(Alarm alarm) {...};
alarm
( Alarm )
Undocumented.

getAll

chrome.experimental.alarms.getAll(function callback)

Gets an array of all the alarms.

Parameters

callback
( function )
Undocumented.

Callback function

The callback parameter should specify a function that looks like this: 

function(array of Alarm alarms) {...};
alarms
( array of Alarm )
Undocumented.

Events

onAlarm

chrome.experimental.alarms.onAlarm.addListener(function(string name) {...});

Fired when an alarm has expired. Useful for transient background pages.

Listener parameters

name
( optional string )
The name of the alarm that has expired.

Types

Alarm

( object )
Undocumented.
name
( string )
Name of this alarm.
delayInSeconds
( integer )
Original length of time in seconds after which the onAlarm event should fire.
repeating
( boolean )
True if the alarm repeatedly fires at regular intervals, false if it only fires once.
